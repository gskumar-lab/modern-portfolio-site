CI/CD Pipeline with AWS & GitHub Actions

  • Personal Project
  • 2024–2025
  • View on GitHub
  • Live Demo
  • Tech Stack: GitHub Actions, AWS EC2, Docker, ECR, CodeDeploy, Flask, NGINX

Overview

This project demonstrates a production-ready CI/CD pipeline using GitHub Actions and AWS services to automate the deployment of a containerized Flask web application.

The pipeline performs automated testing, builds a Docker image, pushes it to Amazon ECR, and deploys it to an EC2 instance via AWS CodeDeploy. NGINX is configured as a reverse proxy for serving the application securely.

This setup emphasizes zero-downtime deployment, modular pipeline stages, and version-controlled infrastructure — core DevOps principles that support agility and scalability in real-world projects.
